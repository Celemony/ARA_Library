std::string getNameForKeySignature (const ARAContentKeySignature &keySignature) const
 Get the user-readable name of keySignature. Will always try to generate a name based on the abstract key signature data and ignore ARAContentKeySignature.name. Only covers a commonly used subset of the key signatures that the abstract key signature data can describe - will return empty string if no suitable name can be provided. If usesASCIISymbols() is false, the returned std::string may contain ARAUtf8Char (potentially multi-byte) symbols.
 
std::string getRootNoteNameForKeySignature (const ARAContentKeySignature &keySignature) const
 Returns the name of the root note of keySignature (useful if getNameForKeySignature() fails). If usesASCIISymbols() is false, the returned std::string may contain ARAUtf8Char (potentially multi-byte) symbols.
